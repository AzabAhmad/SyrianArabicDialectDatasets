This dataset was developed for sentiment analysis in **Syrian Arabic Dialect**.
It contains 20K comments collected from Facebook as it is the most widely used social media platform among the Syrian community. The data is manually annotated into three sentiment classes: P (Positive), G (Negative), and N (Neutral).
